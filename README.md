# 注音聲辨王 (Bopomofo Phonetic Master)

一個精美的互動式注音符號聲辨（聽音辨字）網頁遊戲，提供六選一的趣味挑戰，支援無限關卡持續練習！

## 🌟 遊戲特點
- 🔊 **聽音辨字挑戰**：利用瀏覽器原生語音合成發音，進行六選一聽力測驗。
- ⚡ **鍵盤快捷鍵**：支援 `1`~`6` 選擇答案、`Space`/`Enter` 下一題、`R` 重聽、`H` 提示。
- 📊 **即時統計看板**：追蹤答對題數、當前連勝紀錄、最高連勝紀錄以及答題準確率。
- 🛠️ **豐富遊戲設定**：
  - **範圍選擇**：可設定出題範圍為「全部」、「聲母」、「介母」或「韻母」。
  - **發音速度調整**（0.5x ~ 1.5x）。
  - **發音語音庫選擇**（可切換系統內不同的繁體中文發音）。
  - **例字提示**（例如：播放「ㄅ」時讀出「包子的包」）。
  - **自動下一題**（答對後自動切換，流暢學習）。
- 🎨 **現代化視覺設計**：支援深/淺色主題切換，搭配毛玻璃與繽紛的答對五彩碎紙動畫。
- 📒 **注音對照表與答題紀錄**：隨時打開對照表複習 37 個注音發音與拼音，並能瀏覽本輪答題歷史。

## 🚀 本機運行

在終端機中，切換至本專案目錄並執行以下指令啟動本機伺服器：

```bash
npm run dev
```

啟動後，瀏覽器將會自動開啟：
[http://127.0.0.1:3000](http://127.0.0.1:3000)

## 📁 檔案結構

- `index.html` - 遊戲結構與主畫面
- `style.css` - 美化樣式表與動態效果
- `app.js` - 注音資料、發音控制與遊戲主程式邏輯
- `package.json` - 專案宣告與執行指令腳本
